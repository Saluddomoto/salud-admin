## 1. データベース設計

### 1-1. ER図（概念）

```
users ──────< event_attendees >────── events
                                          │
                                          ├── event_categories
                                          ├── customers
                                          ├── projects
                                          └── tasks
```

---

### 1-2. テーブル定義

#### `events`（予定）

| カラム名 | 型 | NULL | 説明 |
|---|---|---|---|
| `id` | VARCHAR(26) | NO | ULID（主キー） |
| `title` | VARCHAR(255) | NO | 件名 |
| `description` | TEXT | YES | メモ |
| `category_id` | VARCHAR(26) | NO | 予定区分 → `event_categories.id` |
| `start_at` | DATETIME | NO | 開始日時（UTC） |
| `end_at` | DATETIME | NO | 終了日時（UTC） |
| `is_all_day` | BOOLEAN | NO | 終日フラグ |
| `location` | VARCHAR(255) | YES | 場所 |
| `online_url` | VARCHAR(512) | YES | オンラインURL（Zoom / Meet等） |
| `customer_id` | VARCHAR(26) | YES | 顧客 → `customers.id` |
| `project_id` | VARCHAR(26) | YES | 案件 → `projects.id` |
| `task_id` | VARCHAR(26) | YES | タスク → `tasks.id` |
| `subsidy_id` | VARCHAR(26) | YES | 補助金案件 → `subsidies.id` |
| `priority` | ENUM('high','medium','low') | NO | 優先度 |
| `status` | ENUM('scheduled','completed','cancelled','postponed') | NO | ステータス |
| `is_private` | BOOLEAN | NO | 個人予定フラグ（true=本人のみ閲覧可） |
| `is_auto_generated` | BOOLEAN | NO | 自動生成フラグ（補助金期限・タスク期限等） |
| `source` | ENUM('manual','task','subsidy','google','outlook') | NO | 生成元 |
| `external_id` | VARCHAR(255) | YES | 外部カレンダーID（Google等との同期用） |
| `color` | VARCHAR(7) | YES | 表示色（HEX）。NULLの場合はカテゴリ色を使用 |
| `created_by` | VARCHAR(26) | NO | 作成者 → `users.id` |
| `created_at` | DATETIME | NO | 作成日時 |
| `updated_at` | DATETIME | NO | 更新日時 |
| `deleted_at` | DATETIME | YES | 論理削除日時 |

**インデックス**
- `(start_at, end_at)` — カレンダー範囲検索
- `(customer_id)` — 顧客別絞り込み
- `(project_id)` — 案件別絞り込み
- `(created_by)` — 担当者別絞り込み

---

#### `event_categories`（予定区分マスタ）

| カラム名 | 型 | NULL | 説明 |
|---|---|---|---|
| `id` | VARCHAR(26) | NO | 主キー |
| `name` | VARCHAR(50) | NO | 区分名 |
| `color` | VARCHAR(7) | NO | 表示色（HEX） |
| `icon` | VARCHAR(50) | YES | アイコン名 |
| `sort_order` | INT | NO | 表示順 |
| `is_system` | BOOLEAN | NO | システム固定（削除不可） |

**初期データ**

| 商談 | `#0ea5e9` | true |
| 訪問 | `#10b981` | true |
| オンライン | `#8b5cf6` | true |
| 社内会議 | `#64748b` | true |
| 電話 | `#f59e0b` | true |
| 申請期限 | `#ef4444` | true |
| 実績報告 | `#f97316` | true |
| その他 | `#94a3b8` | true |

---

#### `event_attendees`（参加者）

| カラム名 | 型 | NULL | 説明 |
|---|---|---|---|
| `id` | VARCHAR(26) | NO | 主キー |
| `event_id` | VARCHAR(26) | NO | → `events.id` |
| `user_id` | VARCHAR(26) | YES | 社内ユーザー → `users.id` |
| `name` | VARCHAR(100) | YES | 外部参加者名（userがない場合） |
| `email` | VARCHAR(255) | YES | 外部参加者メール |
| `role` | ENUM('organizer','attendee') | NO | 主催者 or 参加者 |
| `response` | ENUM('pending','accepted','declined') | NO | 出欠回答 |

---

#### `event_reminders`（リマインダー）

| カラム名 | 型 | NULL | 説明 |
|---|---|---|---|
| `id` | VARCHAR(26) | NO | 主キー |
| `event_id` | VARCHAR(26) | NO | → `events.id` |
| `minutes_before` | INT | NO | 何分前に通知（例: 30, 60, 1440=1日前） |
| `channel` | ENUM('app','email','line','slack') | NO | 通知チャネル |
| `sent_at` | DATETIME | YES | 送信済み日時 |

---

#### `event_files`（添付ファイル）

| カラム名 | 型 | NULL | 説明 |
|---|---|---|---|
| `id` | VARCHAR(26) | NO | 主キー |
| `event_id` | VARCHAR(26) | NO | → `events.id` |
| `file_name` | VARCHAR(255) | NO | ファイル名 |
| `file_url` | VARCHAR(512) | NO | 保存先URL |
| `file_size` | INT | NO | バイト数 |
| `uploaded_by` | VARCHAR(26) | NO | → `users.id` |
| `uploaded_at` | DATETIME | NO | アップロード日時 |

---

#### `user_calendar_settings`（ユーザーカレンダー設定）

| カラム名 | 型 | NULL | 説明 |
|---|---|---|---|
| `user_id` | VARCHAR(26) | NO | 主キー → `users.id` |
| `default_view` | ENUM('month','week','day') | NO | デフォルト表示 |
| `work_start_hour` | INT | NO | 業務開始時刻（例: 9） |
| `work_end_hour` | INT | NO | 業務終了時刻（例: 18） |
| `show_weekends` | BOOLEAN | NO | 土日表示 |
| `google_token` | TEXT | YES | Google OAuth token（暗号化） |
| `outlook_token` | TEXT | YES | Outlook OAuth token（暗号化） |

## 2. API設計

### 2-1. エンドポイント一覧

#### イベントCRUD

| メソッド | パス | 説明 |
|---|---|---|
| `GET` | `/api/events` | 予定一覧（期間・担当者・カテゴリ等でフィルタ） |
| `POST` | `/api/events` | 予定作成 |
| `GET` | `/api/events/:id` | 予定詳細 |
| `PATCH` | `/api/events/:id` | 予定更新（部分更新） |
| `DELETE` | `/api/events/:id` | 予定削除（論理削除） |
| `PATCH` | `/api/events/:id/move` | ドラッグ&ドロップ日時変更専用（軽量） |

#### 参加者・ファイル

| メソッド | パス | 説明 |
|---|---|---|
| `POST` | `/api/events/:id/attendees` | 参加者追加 |
| `DELETE` | `/api/events/:id/attendees/:userId` | 参加者削除 |
| `POST` | `/api/events/:id/files` | ファイル添付 |
| `DELETE` | `/api/events/:id/files/:fileId` | ファイル削除 |

#### ビュー集計（ダッシュボード用）

| メソッド | パス | 説明 |
|---|---|---|
| `GET` | `/api/events/today` | 今日の予定 |
| `GET` | `/api/events/week` | 今週の予定 |
| `GET` | `/api/events/upcoming-deadlines` | 期限間近リスト |
| `GET` | `/api/events/summary` | 今月の件数サマリ |

#### カテゴリ・設定

| メソッド | パス | 説明 |
|---|---|---|
| `GET` | `/api/event-categories` | カテゴリ一覧 |
| `POST` | `/api/event-categories` | カテゴリ追加（管理者のみ） |
| `PATCH` | `/api/event-categories/:id` | カテゴリ編集 |
| `DELETE` | `/api/event-categories/:id` | カテゴリ削除（is_system=falseのみ） |
| `GET/PUT` | `/api/users/me/calendar-settings` | 自分のカレンダー設定 |

#### 外部連携（将来）

| メソッド | パス | 説明 |
|---|---|---|
| `GET` | `/api/integrations/google/auth` | Google OAuth開始 |
| `GET` | `/api/integrations/google/callback` | Googleコールバック |
| `POST` | `/api/integrations/google/sync` | Google → 手動同期 |
| `GET` | `/api/integrations/outlook/auth` | Outlook OAuth開始 |

### 2-2. リクエスト/レスポンス例

#### `GET /api/events` クエリパラメータ

```
start=2026-06-01T00:00:00Z    // 期間開始（必須）
end=2026-06-30T23:59:59Z      // 期間終了（必須）
user_ids=u1,u2                // 担当者絞り込み（複数可）
category_ids=c1,c2            // カテゴリ絞り込み
customer_id=cust1             // 顧客絞り込み
project_id=proj1              // 案件絞り込み
include_private=true          // 個人予定を含む（自分のみ）
q=商談                        // キーワード検索
```

#### `POST /api/events` リクエストBody

```json
{
  "title": "ものづくり補助金 第17回 申請相談",
  "category_id": "cat_consult",
  "start_at": "2026-07-01T10:00:00Z",
  "end_at": "2026-07-01T11:00:00Z",
  "is_all_day": false,
  "location": "株式会社山田製作所 会議室",
  "online_url": null,
  "customer_id": "cust_yamada",
  "project_id": "proj_monodukuri_17",
  "priority": "high",
  "status": "scheduled",
  "is_private": false,
  "description": "申請書類の最終確認",
  "attendee_user_ids": ["user_tanaka", "user_sato"],
  "reminders": [
    { "minutes_before": 60, "channel": "app" },
    { "minutes_before": 1440, "channel": "email" }
  ]
}
```

#### `PATCH /api/events/:id/move` リクエストBody（D&D専用）

```json
{
  "start_at": "2026-07-02T10:00:00Z",
  "end_at": "2026-07-02T11:00:00Z"
}
```

## 3. 画面一覧

### SC-01: カレンダーメイン画面

**説明**: スケジュール管理のメイン画面。月/週/日/担当者別を切り替えられる。

**主な要素**
- ビュー切り替えタブ（月/週/日/担当者別）
- 今日ボタン・前後ナビゲーション
- フィルターバー（担当者・カテゴリ・顧客・案件）
- カレンダーグリッド
- 予定追加ボタン（+）
- ミニカレンダー（サイドバー）
- 今日の予定サマリ（サイドバー）

---

### SC-02: 予定詳細モーダル（クイックビュー）

**説明**: カレンダー上の予定クリックで表示。詳細情報と操作ボタン。

**主な要素**
- 件名・カテゴリ・日時・場所・URL
- 参加者アバター
- 顧客/案件/補助金へのリンク
- 編集・削除・完了ボタン

---

### SC-03: 予定登録/編集フォーム（モーダルまたはサイドパネル）

**説明**: 予定の新規作成・編集フォーム。

**主な要素**
- 全登録項目（件名/日時/終日/場所/URL/担当者/参加者/顧客/案件/補助金/区分/優先度/ステータス/メモ/添付）
- スマート入力（顧客選択で過去の予定を参考表示）
- リマインダー設定
- 個人/共有 切り替え

---

### SC-04: 予定詳細ページ（フルページ）

**説明**: モーダルから「詳細を見る」で遷移。関連情報をすべて表示。

**主な要素**
- 全項目表示
- 添付ファイル一覧
- 活動ログ（変更履歴）
- 関連案件のカンバン上での現在位置
- 参加者の出欠状況

---

### SC-05: 担当者別カレンダー（チームビュー）

**説明**: 社員ごとの予定を横並びで表示。リソース管理用。

**主な要素**
- 社員選択（複数）
- 日/週単位のタイムライン表示
- 空き時間の自動ハイライト

---

### SC-06: 今日の予定（ダッシュボード ウィジェット）

**説明**: ダッシュボードに埋め込み表示。

**主な要素**
- 今日の予定リスト（時刻順）
- 明日の予定（3件まで）
- 期限間近アラート
- 今月の予定件数

---

### SC-07: 案件詳細内 予定タブ

**説明**: 案件詳細ページの「予定」タブ。案件に紐づく予定一覧と新規登録。

**主な要素**
- 案件に紐づく予定の時系列リスト
- 「面談登録」「商談登録」「締切登録」「訪問登録」ボタン
- カレンダーに移動リンク

---

### SC-08: 顧客詳細内 予定タブ

**説明**: 顧客詳細ページの「予定」タブ。過去・今後の予定を管理。

**主な要素**
- 今後の予定（昇順）
- 過去の面談・訪問履歴（降順）
- 訪問頻度グラフ

---

### SC-09: カレンダー設定ページ

**説明**: ユーザーごとのカレンダー設定。

**主な要素**
- デフォルトビュー選択
- 業務時間設定
- 土日表示設定
- Googleカレンダー連携（将来）
- Outlook連携（将来）
- 通知設定
