# Salud 基幹システム 開発ロードマップ

> 更新日: 2026-07-10
> 本番: https://salud-admin-azure.vercel.app

## ✅ 稼働中の機能

| 機能 | 内容 |
|---|---|
| 顧客・案件・タスク管理 | 一覧・詳細・編集・削除・連絡先管理 |
| 受信トレイ | LINE受信の一覧・要返信フラグ・返信済み管理 |
| LINE連携 | 公式LINE「補助金の窓口」で受信、顧客の自動識別 |
| LINEから予定登録 | 「予定 7/10 14:00 山田製作所 打ち合わせ」で自動登録 |
| 毎朝のLINEダイジェスト | 毎朝 7:30 に今日の予定・期限タスク・要返信数を配信（`digest_enabled`で個別ON/OFF可） |
| Googleカレンダー同期 | 選択した仕事用カレンダーのみ自動同期。OAuth本番公開済み（7日失効リスク解消） |
| スケジュール画面 | 週表示・予定追加・Google同期・担当者ごとの色分け表示 |
| ダッシュボード | KPI・要返信・期限アラート・今日の予定 |
| 設定画面 | メンバー招待（アカウント作成＋初期パスワード発行）・パスワード変更・通知設定の保存 |
