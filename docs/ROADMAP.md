# Salud 基幹システム 開発ロードマップ

> 更新日: 2026-07-07
> 本番: https://salud-admin-azure.vercel.app

## ✅ 稼働中の機能

| 機能 | 内容 |
|---|---|
| 顧客・案件・タスク管理 | 一覧・詳細・編集・削除・連絡先管理 |
| 受信トレイ | LINE受信の一覧・要返信フラグ・返信済み管理 |
| LINE連携 | 公式LINE「補助金の窓口」で受信、顧客の自動識別 |
| LINEから予定登録 | 「予定 7/10 14:00 山田製作所 打ち合わせ」で自動登録 |
| 毎朝のLINEダイジェスト | 毎朝 7:30 に今日の予定・期限タスク・要返信数を配信 |
| Googleカレンダー同期 | 選択した仕事用カレンダーのみ自動同期 |
| スケジュール画面 | 週表示・予定追加・Google同期 |
| ダッシュボード | KPI・要返信・期限アラート・今日の予定 |

## 🚀 次にやる（すぐ着手できる）

| # | 機能 | 内容 | 目安 |
|---|---|---|---|
| 1 | メッセージ→案件・タスク化 | 受信トレイに「タスクにする」「案件にする」ボタン。内容自動入力・顧客自動選択・元メッセージへのリンク | 実装 30〜40分 |
| 2 | メンバー追加 | 佐藤さん・田中さん等のログインアカウント作成 + LINE紐づけ（ダイジェストが全員に届くようになる） | 15分 + 名前・メールの提供待ち |
| 3 | 顧客LINE紐づけ | 顧客からのLINEに自動で社名が付く | 1件 2〜3分（顧客からの受信待ち） |
